# Projet de Prédiction Immobilière avec K-means

## Description
Application interactive de prédiction immobilière utilisant K-means pour la segmentation du marché et Random Forest pour la prédiction des prix.

## Installation
1. Cloner le projet
2. Exécuter `./deploy.sh`
3. Ouvrir http://localhost:8501

## Structure
- `app.py` : Application principale Streamlit
- `notebooks/` : Analyses exploratoires
- `src/` : Code source Python
- `data/` : Données immobilières
- `models/` : Modèles entraînés
